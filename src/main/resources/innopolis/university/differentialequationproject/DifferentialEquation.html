$$\text{Differential equation: }$$ $$\begin{cases}y' = \frac{\sqrt{y-x}}{\sqrt{x}} + 1\\y(1)=10 \\ x \in (1;10)\end{cases}\\ $$ $$\text{Solution: }$$ $$\begin{cases}y = x(\frac{1}{C^2x} - \frac{2}{C\sqrt{x}}+2) \text{, where } x > 0, C \ne 0 \\C_1 = \frac{1}{4} \text{, } C_2 = -\frac{1}{2}\end{cases}\\$$
